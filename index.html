Asistente de Tareas Diarias con Priorización Automática

Un gestor académico integral para la organización de tareas y horarios de clases

Resumen del Proyecto

Este proyecto consiste en una aplicación desarrollada en Python con PyQt5 que permite a los estudiantes gestionar sus tareas académicas y horarios de clase de manera integrada. El sistema ofrece una interfaz intuitiva para registrar tareas con diferentes niveles de prioridad, organizar el horario semanal de clases y visualizar todos los compromisos en un calendario unificado.

Objetivo Principal

Centralizar la gestión académica en una sola herramienta que permita a los estudiantes organizar eficientemente sus tareas, fechas de entrega y horarios de clase.

Problema que Resuelve

Dificultades en la gestión académica que generan conflictos entre entregas de trabajos y clases programadas, causando retrasos y desorganización.

Público Objetivo

Estudiantes universitarios que necesitan una herramienta integral para organizar sus actividades académicas y mejorar su productividad.

Funcionalidades Principales

Gestión de Tareas

  • Registro de tareas con nombre, fecha y hora de entrega
  • Asignación de niveles de prioridad (Baja, Media, Alta)
  • Marcado de tareas completadas
  • Filtrado por fecha, prioridad y estado de completado
  • Ordenamiento automático por prioridad

Gestión de Horario

  • Registro de clases con nombre, NRC, profesor y aula
  • Asignación de días y horarios específicos
  • Detección automática de conflictos de horario
  • Visualización semanal en formato de tabla
  • Asignación de colores para diferenciar materias

Calendario Unificado

  • Visualización integrada de tareas y clases
  • Resaltado de fechas con tareas pendientes
  • Coloración según nivel de prioridad
  • Navegación por fechas para ver eventos específicos

Gestión de Usuarios

  • Sistema multi-usuario
  • Creación y eliminación de perfiles
  • Almacenamiento persistente de datos por usuario
  • Datos guardados en archivos JSON

Interfaz Gráfica

Diseño con Pestañas

La aplicación está organizada en tres pestañas principales para una navegación intuitiva:

  1. Gestión de Tareas: Para agregar, editar y eliminar tareas académicas
  2. Horario de Clases: Para administrar el cronograma semanal de clases
  3. Calendario Unificado: Para visualizar de manera integrada todos los compromisos

Elementos de Interfaz

  • Formularios intuitivos para entrada de datos
  • Tablas interactivas para visualización de horarios
  • Calendario con resaltado de fechas importantes
  • Listas con elementos seleccionables y acciones contextuales
  • Cuadros de diálogo para confirmación de acciones
  • Sistema de alertas y mensajes de retroalimentación

Estructura del Código

Clases Principales

  • TareaManager: Clase principal que gestiona la ventana principal y coordina todas las funcionalidades
  • EditarClaseDialog: Diálogo para agregar o editar información de clases
  • DetailedCaseDialog: Diálogo para mostrar detalles completos de una clase
  • LoginDialog: Diálogo para selección, creación y eliminación de usuarios
